JSDS Documentation!
===================

**JSDS**, Judiciary Statistical Dashboard System is a case management system for
managing cases in an online repository. This webpage provides a guide to documenting
case business process. The documentation splits processes into two process details of:

* :doc:`As-Is <processes/as-is>` process
* :doc:`To-Be <processes/to-be>` process
